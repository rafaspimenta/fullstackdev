HTML Reference

Ordered List element

Tag Name Description
<ol> Ordered List Represents an ordered list of items - typically rendered as a numbered list.
<li> List Item Represent an item in a list. It must be contained in a parent element.

Atributes

Atribute Name Description
reversed Reversed This Boolean attribute specifies that the list's items are in reverse order. Items will be numbered from high to low.
start Start

Sets the numbering type:

  • a for lowercase letters

  • i for lowercase Roman numerals

  • I for uppercase Roman numerals

  • 1 for numbers (default)
